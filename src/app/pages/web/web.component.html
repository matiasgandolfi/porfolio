Web


En pocas palabras, tenemos que los sitios web son un conjunto de páginas web que se relacionan de una u otra forma,
y a los que se puede acceder a través de internet. También se les puede conocer como portal, o website en inglés.

Tipos de sitios web

  • E-commerce
  • Sitios de marca
  • Sitios de comunidades
  • Wikis
  • Blogs

Entre muchos benefcios, el tener una sito web le permitira:

  • Aumenta el alcance de tu negocio
  • Mejora la imagen de tu marca
  • Aumenta tus ventas
  • Te permite vender en línea
  • Mejora el servicio al cliente
  • Posicionamiento en buscadores
